# tilt-logger
Script that logs bluetooth TILT Hydrometer data in range

Needed this for my personal use. Feel free to fork and develop further.

Start write_log_tilt.py in the background to collect data. For example using linux command screen, or a systemd unit.

Read data using display_data.py and display_fg.py.
